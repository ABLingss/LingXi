================================================================================
  STOCK JSON CLIPPER V1.0 — 股票数据一键JSON提取器
  "零安装、纯本地、剪贴板驱动" 的A股数据桥梁
================================================================================

【产品简介】
  在通达信等股票软件中复制代码 → 自动生成AI可分析的结构化JSON → 粘贴到AI对话框
  传统选股工具 + 大模型智能分析 = 工作流闭环

【使用方法】

  1. 双击运行 StockJSONClipper.exe
     → 系统托盘出现图标（屏幕右下角）
     → 程序开始监控剪贴板

  2. 在通达信/同花顺中复制股票代码（Ctrl+C）
     → 程序自动识别、拉取数据
     → 弹出提示 "✅ 000001 已复制到剪贴板（250条）"
     → 直接粘贴（Ctrl+V）到 AI 对话框即可

  3. 右键托盘图标可：
     - 打开信息面板（查看历史记录、调整设置）
     - 清空缓存
     - 退出程序

【代码复制规则】

  ┌──────────────┬─────────────────────────────────┐
  │ 复制内容      │ 含义                            │
  ├──────────────┼─────────────────────────────────┤
  │ 000001       │ 深市平安银行 — 日线数据（默认250条）│
  │ W:000001     │ 周线数据                         │
  │ M:000001     │ 月线数据                         │
  │ #000001      │ 保存全量历史数据为本地JSON文件     │
  │ #W:000001    │ 保存周线全量数据到本地文件         │
  └──────────────┴─────────────────────────────────┘

【输出内容】
  JSON 包含：
  - meta:    股票名称、行业、PE(TTM)、总市值、日期范围
  - indicators: MA5/10/20/60、MACD(DIF/DEA/柱)、RSI(6/12)、BOLL(上/中/下轨)
  - summary: 区间涨跌幅、最高/最低收盘价、平均成交量、年化波动率
  - data:    原始K线数据（日期、OHLCV）

【公式辅助】
  1. 打开面板（右键托盘 → 显示面板）
  2. 在"公式辅助"框中粘贴你的通达信选股公式
  3. 点击"生成Prompt"
  4. 粘贴到AI对话框 → AI会结合当前股票数据逐条判断条件

【配置】
  配置文件: config.ini（自动生成在程序同目录）
  可调整: 默认数据条数、轮询间隔、缓存时间、请求超时

【常见问题】

  Q: 复制代码后无反应？
  A: 确认复制的文本仅为6位数字（如 000001），不含任何其他文字。

  Q: 提示"数据拉取超时"？
  A: 检查网络连接。本工具需要联网访问东方财富公开API。

  Q: 杀毒软件报毒？
  A: 本工具为PyInstaller打包的单文件程序，部分杀软会误报。
     请将 StockJSONClipper.exe 添加到白名单。

  Q: 如何彻底退出？
  A: 右键托盘图标 → 退出。关闭面板不会退出程序。

【技术信息】
  数据源: 东方财富公开API（前复权）
  技术栈: Python 3.9+ / requests / pystray / pywebview / PyInstaller
  依赖: 仅需Windows 10/11，无需安装Python或任何运行环境
